## Как использовать

Вывести словарь со значениями уплаченных компаниями налогов: 

```python 
from pprint import pprint
from paytax import from_file, is_nil

FILENAME = '361db.xml'
xs = [x for x in from_file(FILENAME) if not is_nil(x)]
pprint(xs)
```
